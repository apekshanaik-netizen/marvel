# Introduction to Machine Learning & Data Preparation

Machine Learning (ML) is a way of using data to build models that can make predictions. In the StatQuest video, this is explained using simple visuals like fitting a line through data points. The goal is to find a model that best describes the relationship between variables.

## Key Concepts in Machine Learning

The video mainly focuses on **supervised learning**, where we already know the correct answers (labels). The model learns a relationship between input (like study hours) and output (like exam scores).

A key idea is **error**, which is the difference between the predicted value and the actual value. Machine learning works by finding a model that minimizes this error.

Another important concept is **overfitting**. This happens when the model fits the training data too closely, including noise, and does not perform well on new data.

## Data Preparation Process

The AltexSoft video explains that raw data is not directly usable and must be prepared first.

The process includes:

- **Data collection** from different sources
- **Data cleaning** (handling missing values, removing duplicates, fixing errors)
- **Data transformation** (scaling or normalizing values)
- **Feature engineering** (selecting or creating useful features)

Finally, the dataset is split into **training and testing sets**. The training set is used to build the model, and the testing set is used to evaluate how well it works on new data.
